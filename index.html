Humanitarian Tech

AI-Powered Minefield Documentation System

Document understanding system supporting humanitarian demining operations in Bosnia and Herzegovina.

Challenge

Minefield records contain critical handwritten data that must be digitized to support demining operations. Manual transcription was slow, error-prone, and created bottlenecks in life-saving work.

Solution

We developed an AI-powered system combining YOLO for object detection with custom TensorFlow models for handwritten text recognition. The FastAPI backend processes documents in real-time.

Results

95% Recognition accuracy
10hrs Saved per batch
1,000+ Docs processed
Python FastAPI YOLO TensorFlow OpenCV
